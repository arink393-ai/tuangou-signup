還差一步:填入 Supabase 金鑰

用編輯器打開這個 HTML,把最上面 <script> 裡的兩行改成你的 Supabase 專案資料,存檔後重新整理即可:

const SUPABASE_URL = "https://xxxx.supabase.co";
const SUPABASE_ANON_KEY = "你的 anon public key";

兩個值都在 Supabase 後台 → Project Settings → API 裡。還沒建資料表的話,先到 SQL Editor 跑我提供的建表 SQL。
